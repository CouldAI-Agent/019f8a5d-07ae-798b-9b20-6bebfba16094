# ದೇವರ ವಾಕ್ಯ (Psalm 91:4)

ಈ ಆಪ್ ಕೀರ್ತನೆಗಳು 91:4 ("ಆತನು ನಿನ್ನನ್ನು ತನ್ನ ರೆಕ್ಕೆಗಳಿಂದ ಹೊದಗಿಸುವನು; ಆತನ ಪಕ್ಕಗಳ ಮರೆಯನ್ನು ಆಶ್ರಯಿಸಿಕೊಳ್ಳುವಿ. ಆತನ ಸತ್ಯತೆಯೇ ನಿನಗೆ ಖೇಡ್ಯವೂ ಗುರಾಣಿಯೂ ಆಗಿದೆ.") ವಾಕ್ಯವನ್ನು ಆಧಾರವಾಗಿಟ್ಟುಕೊಂಡು ರಚಿಸಲಾಗಿದೆ. ಇದು ನಮಗೆ ಪ್ರತಿದಿನ ಭರವಸೆ ಮತ್ತು ಧೈರ್ಯವನ್ನು ನೀಡುವ ಉದ್ದೇಶದಿಂದ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿದೆ.

## ವೈಶಿಷ್ಟ್ಯಗಳು
* ಸುಂದರವಾದ ಹಿನ್ನೆಲೆ ಮತ್ತು ಸ್ಪಷ್ಟವಾದ ಅಕ್ಷರಗಳಲ್ಲಿ ದೇವರ ವಾಕ್ಯ.
* ಸರಳ ಮತ್ತು ಆಕರ್ಷಕವಾದ ವಿನ್ಯಾಸ.
* ಮೊಬೈಲ್, ಟ್ಯಾಬ್ಲೆಟ್ ಮತ್ತು ಡೆಸ್ಕ್‌ಟಾಪ್‌ಗೆ ಹೊಂದುವಂತಹ (Responsive) ವಿನ್ಯಾಸ.

## ತಂತ್ರಜ್ಞಾನ
* ಫ್ಲಟರ್ (Flutter)
